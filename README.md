# Online Store Client

Online store responsive web client. Developed using React + Redux and Bootstrap. 

## Installation


Requires specific environment variables to run:


```bash
REACT_APP_API_URL=
REACT_APP_PK_STRIPE=
```

Installing dependencies:

```bash
npm install
```

## Usage

For production run:
```bash
npm start
```
